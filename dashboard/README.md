# ETL Script
## Installation

```sh
pip install -r requirements
```
### Environment variables

The following env variables must be accessible to the script:
```sh
DB_HOST
DB_NAME
DB_USER
DB_PASS
DB_PORT
SCHEMA_NAME
```
## Usage

```sh
streamlit run main.py
```
